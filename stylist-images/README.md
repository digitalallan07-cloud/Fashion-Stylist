## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080 (Full HD) or higher
- **Aspect Ratio**: 16:9 or 4:3 for best results
- **File Size**: Optimize images to under 500KB each for faster loading
- **Quality**: High quality, professional fashion photography

## Image Optimization Tips

1. Use tools like TinyPNG or ImageOptim to compress images
2. Convert to WebP format for better performance (optional)
3. Ensure images are well-lit and properly cropped
4. Use consistent color grading across all images
